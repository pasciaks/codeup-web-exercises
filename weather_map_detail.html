Weather Buddy Details

The image is a live view of the current weather, or how 'hot it is' is, if you know what I mean!

NOTE: A Raspberry PI running my own code is configured with a webcam and a web socket that listens for requests for a new image, and upon receiving the socket.io message, it snaps a picture and delivers it as base64 data to the server. This page, requests the latest base64 of said image and displays it. Ideally, a collection of weather-bot Raspberry PI's connected to WI-FI can help share current live weather, or in some cases, a live snapshot of parking lots, retail checkout lines, or even how 'hot it is' at the local bar, club, or pool! The same 'Smart Sign' can be used inside restaurants to help share rotating images that advertise special deals or menu items.

Smart Sign Technology

Note the date/timestamp of this image. If the 'Please wait' image is shown, the webcam is currently taking the photo. In that case, just refresh the page after a few seconds and it will retrieve the latest image taken!
