Imugi - codeup-web-exercises

I created a bash script and some JS code that compares your codeup-web-exercises to that of a chosen classmate who (more likely than not) has completed all the assignments with proper naming, etc. If you click the link that corresponds to your Github username below, you'll see a list of files that I believe you might be missing, either because you didn't complete the exercise or because of file naming or folder location, etc. Remember exact naming, capitalization, etc matters. No guarantees that the source of these comparisons is totally correct, (use at own risk), but hopefully this might help for those of you looking to recheck your assignment work this weekend. No worries either way, but hope this helps if you are using this weekend to investigate your assignments -- They are all going to be due at the end of next week I believe.

Also, this information is as of the last time I ran and published the results - If you update your assignments after that, I'll need to re run the script so it can re-evaluate my comparison. Feel free to DM me in slack and I can do that sometime again if you like.

  • By the way, if your list is missing the following, it might not be a thing. It is likely these are extra files that chosen source/student created/used in his/her assignments.

    "--- --- logo.png", "--- --- do_while.js", "--- js-html.html"

  • Note: Some folder naming might result in additional (false negative) issues, like using (img) as a folder name instead of (images)

  • Regretfully, since one student folder was selected as a source to compare from, deviations of folder naming will appear to be issues where they might not actually be true issues in these cases.
