About

This is a simple weather app that uses the Open Weather Map API to display the current weather conditions for a given city. The app also displays a 5-day forecast for the city.

The app uses the Mapbox API to display a map of the city. The map is interactive and can be zoomed in and out and panned.

The app uses Bootstrap for styling and layout.

The app uses JavaScript to make AJAX calls to the Open Weather Map API and the Mapbox API.

The app uses the following JavaScript libraries:

  • Mapbox
  • Open Weather MAP API

Search with the text form by typing in a City, State, Country.

Click the home button to navigate and get forecast for your current GPS location.

Double click on map to get a draggable, weather search icon.
